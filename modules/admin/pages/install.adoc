[#windows]
== Установка серверного расширения на Windows

. Запустите пакет установки `{dv} ArchiveManagement.msi`.
. Выберите язык пользовательского интерфейса мастера установки.
. В приветственном окне мастера установки нажмите *Далее*.
+
.Мастер установки модуля "{am}"
image::install-server-hello.png[Мастер установки модуля "{am}"]
+
. Примите условия лицензионного соглашения для продолжения установки и нажмите *Установить*.
+
.Условия лицензионного соглашения
image::install-server-license.png[Условия лицензионного соглашения]
+
. Нажмите кнопку *Готово*, чтобы закрыть мастер установки.
. <<web-ext,Установите>> расширение {am} для {wc}а.

[#web-ext]
=== Установка расширения {am} {wc}а

.Чтобы установить расширение {am} {wc}а:
. Запустите пакет установки `{dv} ArchiveManagement for web-client.msi`.
// +
// .Мастер установки расширения {am} {wc}а
// image::install-client-hello.png[Мастер установки расширения {am} {wc}а]
// +
. Примите условия лицензионного соглашения, чтобы продолжить установку.
// +
// .Условия лицензионного соглашения
// image::install-client-license.png[Условия лицензионного соглашения]
// +
. Нажмите на кнопку *Установить* и дождитесь завершения установки модуля.
// +
// .Мастер установки расширения {am} {wc}а
// image::install-client-confirm.png[Мастер установки расширения {am} {wc}а]
// +
. Нажмите на кнопку *Готово*, чтобы закрыть мастер установки.
+
include::install-linux:ROOT:partial$restart-and-db.adoc[]

. Импортируйте решения с помощью панели управления {wc}ом, подробнее см. в разделе "xref:6.1@webclient:admin:control-panel-solutions.adoc[]".
. Для работы с модулем {am} в программе {kvr} выберите разметку меню _DocumentManagement_Nomenclature_.

[#uninstall]
== Удаление модуля

. Чтобы удалить модуль в ОС Linux, выполните команду:
+
[subs=attributes]
 $ sudo apt-get purge docsvision-{wt}
+
. В ОС Windows компоненты модуля удаляются стандартными средствами.
